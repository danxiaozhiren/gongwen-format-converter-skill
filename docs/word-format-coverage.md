# Word-only 覆盖矩阵

本文定义 `format-xingwen-word` 对已有 `.docx` 的当前覆盖边界。项目定位是“行文出手前对照检查事项”场景下的安全保守 Word 版式整理，不是完整 Word 排版引擎，也不承诺跨 Word/WPS/LibreOffice 的全视觉等价。

## 状态说明

| 状态 | 含义 |
| --- | --- |
| `supported` | 脚本会自动处理，且有 fixture 或 unittest 证明。 |
| `partial` | 可处理常见情况，但复杂文档或渲染效果仍需人工确认。 |
| `diagnose_only` | 只识别和报告，不自动修改。 |
| `preserve_only` | 尽量保留原对象和原状态，不主动重写。 |
| `unsupported` | 当前 skill 明确不支持。 |

## 覆盖矩阵

| 范围 | 当前状态 | 当前行为 | 下一步验收目标 |
| --- | --- | --- | --- |
| 纸张大小 | `supported` | 按 checklist 预设设置 A4。 | 多节文档差异报告。 |
| 页边距 | `supported` | 设置上 3.7 cm、下 3.5 cm、左 2.7 cm、右 2.7 cm。 | 多节文档逐节验收。 |
| 版心参考 | `supported` | 报告 156 mm x 225 mm 版心参考。 | 渲染后容差检查。 |
| 文档网格 | `partial` | 写入 `docGrid` 的 22 行/28 字参考并在报告中提示需 Word/WPS 复核。 | PDF/截图层面的行数与版心校验。 |
| 正文段落角色识别 | `partial` | 以候选角色、置信度和 reason code 识别标题、正文、四级标题、附件、落款、日期等常见角色。 | 更多短标题、文号、异常编号样例。 |
| 段落对齐 | `supported` | 按角色设置居中、左对齐、右对齐或两端对齐。 | 覆盖页眉页脚段落。 |
| 缩进 | `supported` | 按角色设置首行、左右缩进。 | 增加悬挂缩进样例。 |
| 行距 | `supported` | 设置标题 36 pt、正文和标题层级 30 pt 固定行距。 | 渲染后每页行数检查。 |
| 段前段后 | `supported` | 按 checklist 预设归零或设置标题后间距。 | 增加标题跨页样例。 |
| 分页控制 | `partial` | 设置孤行控制、与下段同页、段中不分页等安全项。 | 报告分页控制变更。 |
| 大纲级别 | `partial` | 对标题角色写入 outline level。 | 验证导航窗格影响。 |
| 手写编号 | `supported` | 将 `一、`、`（一）`、`1.`、`（1）` 作为既有文本保留并套格式。 | 增加混合编号和异常编号样例。 |
| Word 自动编号 | `preserve_only` | 检测并保留，不重建编号体系。 | 独立设计自动编号重建策略。 |
| 中文字体 | `supported` | 设置东亚字体名称：方正小标宋简体、仿宋_GB2312、黑体、楷体_GB2312、宋体。 | 报告本机缺失字体。 |
| 西文字体 | `partial` | 同步设置 ascii/hAnsi 为 Times New Roman。 | 区分混合语言 run 的更细粒度验证。 |
| 字号 | `supported` | 按角色设置二号、三号、四号。 | 覆盖字符样式继承场景。 |
| 加粗/斜体/下划线 | `partial` | 按 checklist 设定二级标题加粗、其他常规项不加粗；默认黑色。 | 避免误清除正文语义强调。 |
| 表内文字 | `supported` | 默认按正文样式处理表格单元格段落，保持表格文本。 | 区分表头和数字列。 |
| 表格结构 | `partial` | 显式 `--format-tables` 时设置宽度、边框、单元格边距、垂直居中、首行重复。 | 支持复杂合并单元格前置风险判断。 |
| 页码字段 | `partial` | 默认格式化已有 PAGE 字段；显式 `--add-page-numbers` 时添加页码。 | 支持首页不同、奇偶页不同、总页数字段。 |
| 页眉文字 | `preserve_only` | 检测并保留，不默认重写。 | 增加页眉样例。 |
| 页脚文字 | `preserve_only` | 检测并保留，不默认重写。 | 增加页脚样例。 |
| 红头/版头红线 | `unsupported` | 不生成发文机关标志、红线、发文字号版头或签发人位置。 | 需要独立模板和渲染验收。 |
| 图片 | `preserve_only` | 统计 inline shapes，默认不移动不缩放。 | 区分普通图片、印章、二维码。 |
| 印章图片 | `preserve_only` | 作为图片/对象保留。 | 设计锚点、层级、环绕、大小校验策略。 |
| 浮动文本框 | `unsupported` | 只通过 XML 计数诊断，不能可靠编辑。 | 深入 OOXML 读取位置和文本。 |
| 形状/线条 | `unsupported` | 诊断并保留。 | 建立安全读取策略。 |
| 水印 | `unsupported` | 仅作为特殊对象范围提示。 | 识别文字水印和图片水印。 |
| 图表/公式/嵌入对象 | `preserve_only` | 作为复杂对象保留。 | 只做存在性和风险报告。 |
| 批注 | `preserve_only` | 关系存在时报告，不自动改；批注文本纳入内容保持哈希。 | 支持统计批注数量和作者。 |
| 修订 | `preserve_only` | 检测插入/删除标记，不接受或拒绝修订。 | 设计显式“保留/接受/拒绝”模式。 |
| 域代码 | `partial` | 检测 PAGE、NUMPAGES、TOC、DATE、REF、PAGEREF 等字段并报告字段更新风险。 | 支持更多字段类型和可选字段更新工作流。 |
| 目录 | `preserve_only` | 作为 TOC 字段/复杂结构保留，提示格式化后需要更新域。 | 渲染后检查。 |
| 脚注尾注 | `preserve_only` | 关系和引用可诊断，脚注/尾注文本纳入内容保持哈希，不能格式化。 | 设计脚注文本格式化范围。 |
| 书签 | `diagnose_only` | 可统计 bookmarkStart。 | 避免破坏交叉引用和书签范围。 |
| 超链接 | `diagnose_only` | 可统计 hyperlink。 | 设计是否规范样式。 |
| 样式系统 | `diagnose_only` | 统计内置/自定义样式和样式样本。 | 支持样式继承解析和样式冲突报告。 |
| Markdown/纯文本输入 | `unsupported` | CLI 只接受 `.docx`。 | 不属于主 skill。 |
| stdin 输入 | `unsupported` | 不从 stdin 生成 Word。 | 不属于主 skill。 |
| 模板样式提取/套用 | `unsupported` | 不再作为主 skill 能力。 | 如需恢复，另起独立 skill。 |
| 批量处理 | `unsupported` | CLI 一次处理一个 `.docx`。 | 如有需要再设计目录遍历和失败隔离。 |
| 内容保持校验 | `supported` | 对正文、表格、页眉页脚、文本框、批注、脚注和尾注文本生成哈希并对比；失败时拦截正式输出，报告 `requested_output`、`output_withheld`，并返回非零退出码。 | 增加更多独立部件和嵌入对象的可选哈希覆盖。 |
| 渲染级校验 | `partial` | 提供可选 LibreOffice/OpenOffice PDF 导出 smoke check；无渲染器时跳过。 | 增加页面截图、版心/行数和视觉差异验收。 |

## 对外表述边界

推荐表述：

- “只处理已有 `.docx`，按行文检查版式整理 Word 格式。”
- “自动处理常见页面、段落、文字、表格文字、页码字段，并对复杂对象做诊断和保留。”
- “严格视觉效果仍需 Word/WPS 最终确认。”

避免表述：

- “支持 Markdown/纯文本转 Word。”
- “支持模板完整复刻。”
- “完全处理 Word 全格式。”
- “保证与单位模板视觉效果 100% 一致。”
