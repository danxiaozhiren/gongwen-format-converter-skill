正文：

```text
参加「SOLO 技能创作大赛」时，我做了一个很贴近办公场景的 Skill：gongwen-format-converter。

它解决的不是“AI 帮我写正文”，而是另一个更常见的问题：文章已经写好了，但 Word 格式还要反复调。标题字号、正文行距、首行缩进、一级标题、二级标题、页边距、内部简报模板……这些事情单独看都不难，但每次手工处理都很费时间。

这个 Skill 目前有三个模式：

1. 仅格式化：正文不改，只调整页面、字体、字号、行距、缩进、标题层级等。
2. 格式诊断：不改文档，先扫描整篇文档当前格式，输出页面、字体、颜色、下划线、行距、表格图片等诊断报告。
3. 模板复刻：先提取模板格式清单，列出目标文档还有哪些格式没有覆盖，确认后再套用模板。

我特别加了一个隐私友好的设计：处理内部材料时，默认不在报告里复述完整正文；模板复刻只学习格式，不暴露模板内容。

用 Skill 之前：
- 手工调标题、正文、行距、缩进；
- 不确定哪里不符合公文/简报习惯；
- 模板能看但不好复刻。

用 Skill 之后：
- 先诊断格式问题；
- 再确认格式方案；
- 最后生成整理后的 Word 文档。

GitHub 地址：
https://github.com/danxiaozhiren/gongwen-format-converter-skill

#SOLO 技能创作赛
```
