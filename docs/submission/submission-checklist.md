# 参赛提交前检查清单

## 作品本体

- [ ] GitHub 仓库可公开访问。
- [ ] Skill 安装路径可访问：`skills/gongwen-format-converter`。
- [ ] README 面向安装者，包含 Agent 安装和手动下载安装两种方式。
- [ ] README 明确说明官方格式参考依据：《党政机关公文处理工作条例》和 `GB/T 9704-2012`。
- [ ] README 明确说明内容边界：不改写、不补写、不重排正文。
- [ ] Skill 支持三种模式：现文格式化、现文格式诊断、现文模板套用。
- [ ] 本机重新安装或更新后能正常触发。
- [ ] 至少准备一个脱敏测试案例，能展示输入、处理过程和输出。

## 能力验证

- [ ] 现文格式化可输出 `.docx`。
- [ ] 报告包含 `content_preservation`，且能说明文本是否保持不变。
- [ ] 报告包含 `coverage`，能区分 `formatted`、`preserved`、`diagnosed_only`、`not_detected`、`unsupported`、`needs_review`。
- [ ] 报告包含 `format_changes.page`，展示页面、版心、文档网格变化。
- [ ] 报告包含 `format_changes.paragraph_controls`，展示分页控制、大纲级别、编号/项目符号等变化。
- [ ] 页码默认不新增；只有明确使用 `--add-page-numbers` 时才添加。
- [ ] 表格默认保留结构；只有明确使用 `--format-tables` 时才规范结构。
- [ ] 模板套用先输出模板覆盖情况，不直接复制模板正文。
